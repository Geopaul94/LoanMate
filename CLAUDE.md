# LoanMate

EMI & Loan Tracker — Android app.

## Stack
- Kotlin, Jetpack Compose
- Room (persistence), Hilt (DI), WorkManager (background/reminders)

## Conventions
- Clean architecture: data / domain / presentation layers.
- Compose idioms; avoid deprecated APIs.
- Coroutines/Flow for async; never block the main thread.
